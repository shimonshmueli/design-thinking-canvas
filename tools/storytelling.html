Define · tool / method

Storytelling

What it is

Packaging research findings as narrative — a day in the user's life, a journey through the pain point — so the insight lands emotionally, not just intellectually. Teams and stakeholders act on what they feel; a well-told user story creates shared conviction about which problem matters in a way a spreadsheet of findings cannot.

How to use it

  1. Choose the one insight the audience must retain, and build the story around it.
  2. Use a real (anonymized) user and real details from research.
  3. Structure it: situation, struggle, consequence — let the tension carry the point.
  4. Deliver it wherever alignment is needed: kickoffs, stakeholder reviews, the brief itself.

Watch out for

A moving story about an unrepresentative user can steer the whole project toward an edge case. Pair narrative with the pattern evidence behind it.
